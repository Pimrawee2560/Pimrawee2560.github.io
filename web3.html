Complaint System, Nakhon Chum Subdistrict Kamphaeng Phet

For the Software Engineering course project, I developed the Backend and designed all the system functions. It was developed using HTML and PHP. The project's database efficiently utilizes the MySQL database management system, and Apache is used as the web server. The system is designed so that when any incident occurs, citizens who want to file a complaint can report it to the Subdistrict Administrative Organization and track its progress. The Subdistrict Administrative Organization can manage the complaint information; upon receiving a complaint, a notification message will be sent via Line, and the work progress for each task can be updated. Furthermore, the system allows for the evaluation of the performance of units within the Subdistrict Administrative Organization, and the evaluation results are displayed in the form of performance graphs.

Tools:

  • Frontend: HTML, CSS, JavaScript
  • Backend: PHP (for server-side logic)
  • Database: MySQL (for storing and managing data)
  • Database Management: phpMyAdmin (a tool for interacting with the MySQL database)
  • Design Tool: Figma (used for creating the website's visual design)
  • Tool: XAMPP (Simulate a server on a computer),Apache(Is a web server),Line Notify, Visual Studio Code
