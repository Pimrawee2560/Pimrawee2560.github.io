Program for scanning the number of people in an image

A program for scanning the number of people in an image, developed for the Digital Image Processing course project using Python and run on Google Colab. The developed code and the image to be scanned are uploaded to the web environment..

The system should have the following capabilities:

  • Scan human faces
  • Separate human faces in the image
  • Display the number of people in the image

Tools:

  • Programming Language: Python
  • Tool: Google Colab
