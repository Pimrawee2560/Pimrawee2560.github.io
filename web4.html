REG Online Registration System

For the Internet Programming course project, I developed the Frontend and designed the website's interface or UI using HTML, CSS, and JavaScript.

The system should have the following capabilities:

  • Course registration
  • Checking course details, including the instructor and class hours
  • Student login using student ID to access the student portal
  • Faculty/staff login to access the organization portal
  • Administrators can view statistical references as graphs showing the number of students from each faculty, their hometowns, their originating provinces before attending Naresuan University.

Tools:

  • Frontend: HTML, CSS, JavaScript
  • Backend: PHP (for server-side logic)
  • Database: RDBMS (Relational Database Management System)
  • Design Tool: Canva (used for creating the website's visual design)
  • Tool: XAMPP (Simulate a server on a computer),Apache(Is a web server), Visual Studio Code
